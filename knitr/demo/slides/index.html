making HTML5 slides with pandoc and knitr

2012-05-01

Update (2014/12/24): the rmarkdown package has made this much much easier, so you should probably ignore this page. Please see http://rmarkdown.rstudio.com.

Here is a simple demo of making DZslides with knitr and pandoc:

  • Rmd source: knitr-slides.Rmd
  • markdown output: knitr-slides.md

Run the pandoc command mentioned in the slides to get the HTML output. Use Left/Right arrows to navigate through the slides.

Also see slides made by other users in the showcase page, e.g. Gaston Sanchez.
